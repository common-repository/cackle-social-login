Cackle Social Login allows your visitors to login and register with 20+ Social Networks like for example Facebook, VK, Twitter, Google, LinkedIn, etc.


== Description ==

Cackle Social Login plugin for Wordpress allows users to use social networks accounts to login or register in Wordpress.

Supported providers:

Google, Google+, Facebook, Twitter, Vkontakte, Mail.ru, Odnoklassniki, Yandex, Rambler, LinkedIn, Yahoo, MyOpenID, Live Journal, Flickr, WordPress, Blogger, Verisign, Dropbox, Foursquare, Instagram and others.


= Cackle Features =

* Unlimited API login requests
* Checking user email if social network provider doesn't provide it
* Choosing providers to show on special order
* Using with Cackle comments( logged users could leave comments with the same login

Cackle ( http://cackle.me ) social platform have more than 27 000 sites installed their widgets and includes comments, chat, poll, login and review widgets.

== Installation ==

1. Upload  folder `cackle-social-login` to the `/wp-content/plugins/` directory
2. For getting keys, you must be [registered on Cackle](http://cackle.me/account/signup?demo=[{%22type%22:%22comment%22,%22period%22:12}]&lang=en "Cackle")   for a free 5-day trial period
3. Enter your site Id which you can find on the left top corner in Cackle admin panel
4. Please, go to yoursiteurl/wp-login.php to check that social icons appeared
